## 5. 프로젝트 개요

Dishly는 냉장고 속 재료와 개인 식단 목표를 기반으로 AI 레시피를 추천하고, 섭취 기록까지 이어서 관리하는 다이어트 레시피 서비스입니다. 사용자는 보유 재료를 등록하고 목표 칼로리, 조리 시간, 난이도 같은 조건을 선택해 현재 상황에 맞는 레시피를 빠르게 받을 수 있습니다.

- 프로젝트명: Dishly AI Recipe
- 서비스 유형: AI 기반 다이어트 레시피 추천 및 식단 기록 서비스
- 주요 사용자: 식단 관리를 하고 싶지만 매번 메뉴를 고르기 어려운 1인 가구, 다이어터, 냉장고 재료를 효율적으로 쓰고 싶은 사용자
- 핵심 가치: 재료 낭비 감소, 메뉴 결정 시간 단축, 칼로리 중심의 식단 관리

## 6. 문제 정의

식단 관리를 시작한 사용자는 “무엇을 먹을지”를 매번 새로 결정해야 합니다. 냉장고에 어떤 재료가 있는지, 칼로리는 얼마나 되는지, 조리 시간은 충분한지, 오늘 기록과 맞는지까지 동시에 고려해야 하므로 선택 피로가 커집니다.

- 냉장고 속 재료를 제대로 활용하지 못해 식재료가 남거나 버려지는 문제가 있습니다.
- 레시피 검색은 많지만, 내 보유 재료와 목표 칼로리에 딱 맞춘 결과를 찾기 어렵습니다.
- 식단 기록과 레시피 추천이 분리되어 있어 추천 결과가 실제 식단 관리로 이어지기 어렵습니다.
- 모바일에서는 빠른 선택이 중요하지만, 영양 정보와 추천 맥락도 함께 확인되어야 합니다.

김서연은 식단 관리를 시작했지만 바쁜 일상 속에서 재료 확인, 레시피 검색, 칼로리 계산을 따로 하기 번거로워하는 사용자입니다. 퇴근 후 30분 안에 만들 수 있는 건강한 한 끼를 원하고, 냉장고에 남은 재료를 버리지 않고 활용하고 싶어 합니다.

- 목표: 냉장고 재료를 활용해 목표 칼로리에 맞는 메뉴를 빠르게 결정하기

- 페인포인트: 재료는 있지만 어떤 음식을 만들지 모르고, 레시피 검색 결과가 너무 많아 선택이 피곤함
- 니즈: 보유 재료, 조리 시간, 칼로리를 한 번에 반영한 추천과 식단 기록 연결

## 8. 디자인 컨셉

Dishly의 디자인 컨셉은 “가볍게 시작하는 건강한 식단 관리”입니다. 식단 관리 앱이 주는 부담감을 줄이기 위해 밝은 배경, 파란색 중심의 신뢰감 있는 컬러, 음식 이미지 중심의 카드 UI를 사용했습니다.

- 키워드: Fresh, Simple, Smart, Personal
- 컬러: 신뢰감을 주는 블루를 주색으로 사용하고, 식재료와 상태 표현에는 그린·오렌지 계열을 보조색으로 사용합니다.
- 레이아웃: 모바일 하단 탭과 데스크톱 사이드바를 모두 고려한 반응형 구조입니다.
- UI 원칙: 사용자가 직접 입력해야 하는 부담을 줄이고, 버튼·칩·카드 중심으로 빠르게 선택하게 합니다.
- 시각 자산: 실제 음식 이미지와 식재료 아이콘을 활용해 추천 결과를 직관적으로 보여줍니다.

## 9. 정보 구조

- 홈: 현재 칼로리 상태, 추천 레시피, 유통기한 임박 재료, 취향 기반 추천
- 레시피 추천: 재료 추가, 목표 칼로리, 조리 시간, 난이도 선택, 오늘의 추천
- 나의 냉장고: 냉장고 이미지 스캔, 재료 직접 입력
- 레시피 기록: 생성·저장한 레시피와 사용 재료 확인
- 식단 기록: 월별 캘린더, 일일 섭취 칼로리, 식사별 기록
- 마이페이지: 사용자 정보, 만든 레시피, 저장 레시피, 공지/고객센터

## 10. 유저 플로우

Dishly의 유저 플로우는 “재료를 등록하고, 조건을 선택하고, 추천받은 레시피를 저장하거나 식단 기록에 반영하는 흐름”으로 구성됩니다. 추천 결과가 일회성으로 끝나지 않고 사용자의 식단 관리 루프로 이어지는 것이 핵심입니다.

1. 냉장고 재료 등록: 카메라 스캔 또는 직접 입력으로 현재 보유 재료를 등록합니다.
2. 추천 조건 선택: 목표 칼로리, 조리 시간, 난이도를 칩 UI로 빠르게 선택합니다.
3. AI 레시피 추천: 보유 재료와 조건을 기준으로 오늘 만들 수 있는 메뉴를 제안합니다.
4. 레시피 저장 및 확인: 마음에 드는 추천 결과를 저장하고 재료와 칼로리를 다시 확인합니다.
5. 식단 기록 반영: 섭취한 메뉴를 기록해 현재 칼로리와 남은 목표량을 추적합니다.

핵심 UX 포인트는 입력 부담을 줄이고, 추천 결과가 저장과 기록으로 이어지게 만들어 사용자의 반복 사용 이유를 만드는 것입니다.

## 11. 화면 소개

### 홈

홈 화면은 사용자의 현재 상태를 가장 먼저 보여줍니다. 현재 섭취 칼로리와 남은 칼로리를 반원 그래프로 표현하고, 추천 레시피와 유통기한 임박 재료를 이어서 확인할 수 있습니다.

### 레시피 추천

재료를 추가하고 목표 칼로리, 조리 시간, 난이도를 선택해 맞춤 레시피를 추천받는 화면입니다. 칩 형태의 선택 UI를 사용해 빠르게 조건을 조합할 수 있도록 설계했습니다.

### 나의 냉장고

냉장고 이미지를 스캔하거나 직접 입력해서 보유 재료를 등록하는 화면입니다. 카메라 스캔 가이드를 중심에 배치해 사용자가 해야 할 행동을 명확히 보여줍니다.

### 레시피 기록

추천 또는 생성된 레시피를 저장하고 다시 확인하는 화면입니다. 대표 음식 이미지, 칼로리, 포함 재료를 함께 보여줘 실제 조리 전에 필요한 정보를 빠르게 확인할 수 있습니다.

### 식단 기록

월별 캘린더와 일일 칼로리 그래프를 통해 식단 이력을 확인합니다. 아침·점심·저녁 기록을 음식 이미지와 칼로리로 정리해 추적 경험을 단순화했습니다.

## 14. Project 02 · Hamniverse 프로젝트 개요

Hamniverse는 햄스터를 키우는 사용자가 햄스터의 일상 케어, 상태 확인, 햄터뷰, 인테리어, 쇼핑 경험을 하나의 모바일 앱에서 관리할 수 있도록 설계한 반려 햄스터 관리 애플리케이션입니다.

- 프로젝트명: Hamniverse
- 서비스 유형: 반려 햄스터 케어 및 관리 애플리케이션
- 주요 사용자: 햄스터를 처음 키우는 보호자, 반복 케어를 놓치기 쉬운 사용자, 햄스터 용품과 케이지 꾸미기에 관심이 있는 사용자
- 핵심 가치: 케어 루틴 관리, 감성적 애착 형성, 인테리어/쇼핑 연결

## 15. Project 02 · 문제 정의

햄스터 케어는 작고 반복적인 일을 매일 놓치지 않는 것이 중요합니다. 하지만 초보 보호자는 먹이, 물, 청소, 환경 관리, 용품 구매 정보를 각각 따로 확인해야 하며, 관리 행동이 기억에 의존하기 쉽습니다.

- 먹이, 물, 청소 같은 매일의 관리 행동을 놓치기 쉽습니다.
- 햄스터의 컨디션과 필요한 케어 정보를 직관적으로 확인하기 어렵습니다.
- 케이지 인테리어와 용품 정보가 분리되어 있어 초보자가 선택하기 어렵습니다.
- 단순 기록형 관리 앱은 사용자가 매일 들어올 감성적 동기가 부족합니다.

## 16. Project 02 · 디자인 컨셉

햄니버스의 디자인 컨셉은 “한 손의 작은 세상”입니다. 부드러운 베이지 톤, 둥근 카드, 햄스터 캐릭터, 케이지 일러스트를 활용해 관리 앱이지만 부담스럽지 않고 매일 들어오고 싶은 분위기를 만들었습니다.

- 컬러: 폰트색은 짙은 브라운, 메인색은 골드 브라운과 크림, 서브색은 연한 베이지 톤을 사용합니다.
- 폰트: 로고는 둥글고 개성 있는 글꼴, 본문은 LINE Seed 계열의 고딕체, 버튼은 귀여운 동글체 톤을 기준으로 합니다.
- UI 무드: 따뜻하고 아기자기한 반려 햄스터 세계관을 보여줍니다.
- 인터랙션: 메인 체크, 햄터뷰, 인테리어, 쇼핑으로 이어지는 흐름을 단순 관리가 아닌 애착 경험으로 설계합니다.

## 17. Project 02 · 유저 플로우

1. 앱 입장: 사용자가 햄니버스의 작은 세계에 진입합니다.
2. 로그인: 계정으로 접속해 자신의 햄스터 관리 공간에 들어갑니다.
3. 홈 화면: 햄스터 프로필과 하루 체크 항목을 먼저 확인합니다.
4. 햄터뷰: 대화형 화면으로 햄스터의 상태와 케어 팁을 자연스럽게 확인합니다.
5. 인테리어: 현재 케이지와 가구를 확인하고 원하는 아이템을 탐색합니다.
6. 햄쇼핑: 추천 상품과 케어 용품을 확인해 필요한 구매 행동으로 이어집니다.

핵심 UX 포인트는 반복적인 반려동물 관리 행동을 귀엽고 감성적인 경험으로 바꿔 사용자가 매일 앱에 들어올 이유를 만드는 것입니다.

## 18. Project 02 · 화면 소개

### 로그인

서비스의 첫인상을 만드는 화면입니다. 따뜻한 베이지 배경과 햄스터 캐릭터를 사용해 작은 반려동물 세계에 들어오는 느낌을 전달합니다.

### 메인

햄스터 프로필과 하루 체크 항목을 중심으로 구성했습니다. 사용자는 먹이, 물, 청소, 산책 같은 반복 케어를 빠르게 확인할 수 있습니다.

### 햄터뷰
